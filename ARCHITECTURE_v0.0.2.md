## 🗂️ 数据流向

```
┌─────────────────────────────────────────────────────────────┐
│                         数据流                               │
└─────────────────────────────────────────────────────────────┘

用户操作
   │
   ▼
┌──────────────┐
│  UI组件交互   │ (按钮点击、表单输入等)
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ AuthContext  │ 更新认证状态
│  or          │
│ Component    │ 组件内部状态
│  State       │
└──────┬───────┘
       │
       ├─► localStorage ◄─── 持久化
       │   (用户信息)
       │
       └─► 触发重新渲染
             │
             ▼
       ┌──────────────┐
       │   UI更新      │ (显示新状态)
       └──────────────┘

示例流程：安装Skill

1. 用户点击"安装按钮"
   ↓
2. handleInstall() 函数执行
   ↓
3. updateUser({ installedSkills: [..., newSkillId] })
   ↓
4. AuthContext 更新 user 对象
   ↓
5. localStorage 保存新状态
   ↓
6. 组件重新渲染
   ↓
7. UI显示"已安装"状态
   ↓
8. Toast通知 "安装成功!"
```

## 🎯 智能推荐算法

```
┌─────────────────────────────────────────────────────────────┐
│                      推荐算法逻辑                             │
└─────────────────────────────────────────────────────────────┘

输入:
  • user.profession (用户职业)
  • user.installedSkills (已安装列表)
  • mockSkills (全部Skills)

步骤1: 职业匹配推荐
  ├─► 筛选: skill.recommendedProfessions 包含 user.profession
  └─► 结果: professionBasedSkills[]

步骤2: 热门推荐
  ├─► 排序: 按 skill.downloads 降序
  └─► 结果: popularSkills[]

步骤3: 合并去重
  ├─► 合并: [...professionBasedSkills, ...popularSkills]
  ├─► 去重: 基于 skill.id
  └─► 排除: user.installedSkills 中的Skills

步骤4: 限制数量
  └─► 取前4个: recommendedSkills.slice(0, 4)

输出:
  • 最多4个推荐Skills
  • 优先显示职业匹配
  • 其次显示热门Skills

示例:
  用户: AI爱好者
  已安装: [Skill-1, Skill-2]
  
  推荐结果:
    1. Text Summarizer (职业推荐)
    2. Image Processor (职业推荐)
    3. Video Transcriber (职业推荐)
    4. Social Media Manager (热门推荐)
```

## 🔐 认证流程详解

```
┌─────────────────────────────────────────────────────────────┐
│                     认证状态管理                             │
└─────────────────────────────────────────────────────────────┘

AuthContext 状态:
┌─────────────────────┐
│ isAuthenticated     │ 是否已登录
│ user                │ 用户对象 (或 null)
│ isLoading           │ 加载状态
├─────────────────────┤
│ login(user)         │ 登录方法
│ logout()            │ 登出方法
│ updateUser(updates) │ 更新用户信息
└─────────────────────┘

初始化流程:
┌─────────────────┐
│ App启动         │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ AuthProvider    │ useEffect
│   初始化        │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 读取localStorage│
│ key:            │
│ autocraft_user  │
└────────┬────────┘
         │
         ├─► 有数据 → 解析并恢复登录状态
         │            isAuthenticated: true
         │            user: {...}
         │
         └─► 无数据 → 保持未登录状态
                      isAuthenticated: false
                      user: null

登录流程:
┌─────────────────┐
│ 用户提交登录表单 │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 验证通过        │ (验证码或API Key)
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 创建User对象    │
│ {               │
│   id,           │
│   email/apiKey, │
│   userType,     │
│   ...           │
│ }               │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ login(user)     │
│ • 保存到        │
│   localStorage  │
│ • 更新Context   │
│   状态          │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 导航到下一步    │
│ /select-        │
│  profession     │
└─────────────────┘

登出流程:
┌─────────────────┐
│ 用户点击登出    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ logout()        │
│ • 清空          │
│   localStorage  │
│ • 重置Context   │
│   状态          │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 导航到首页      │
│ /               │
└─────────────────┘
```
